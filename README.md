# AI Trading Agent

Autonomous trading agent using reinforcement learning for crypto markets.

## Features
- RL-based decision making
- Multi-timeframe analysis
- Risk management
- Backtesting framework

## Tech Stack
- Python 3.11+
- TensorFlow / PyTorch
- CCXT for exchange integration

## Status
🚧 Research phase - Training models


## Usage

```bash
# Run the script
python main.py

# With custom parameter (if supported)
python main.py <parameter>
```
